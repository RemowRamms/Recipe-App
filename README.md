RECIPE APP
 
A sleek, responsive web application built with React and Tailwind CSS that allows users to search and browse recipes fetched from TheMealDB API.

FEATURES
• Search for recipes by name, category or ingredient.
• Browse recipe details including ingredients, instructions, and images.
• Responsive design ensures a smooth experience across desktop, tablet and mobile.
• Clean UI built with Tailwind CSS, React hooks and components.
• Simple, intuitive navigation and layout.

TECH-STACK
• Front-end: React
• Styling: Tailwind CSS
• Data: TheMealDB API
• Deployment: Netlify (live at https://leafy-druid-66e089.netlify.app)

GETTING STARTED (Local Setup)
• Node.js (v14 or higher)
• npm or yarn

INSTALLATION
# CLONE THE REPO
git clone <your-repo-url>  

# CHANGE INTO THE PROJECT DIRECTORY
cd <project-folder>  
  
# INSTALL DEPENDENCIES
npm install 

# OR 
yarn install  

RUNNING THE APP

# For development mode  
npm start  
# or  
yarn start
